OUTFIT DESIGNS

The design process is based on creating six designs fllowing a theme and a palette, so that there is harmony between a set, but also to increase variety between outfit batches, for example:

Vaporeon Inspired

This set is inspired by the water Pokemon Vaporeon, the palette being the same colors used by this Pockett Monster.

Sunrise Inspired

This set is inspired by the sunrise, ergo the gradient used throughout the outfits.

Sylveon Inspired

This set is inspired by the Pokemon Sylveon, the palette being the same colors used by this Pockett Monster, and an added highligh of using their bow in the design.

Sakura Inspired

This set uses cherry blossom as it's main design element, as well as a vibrant pink palette.

Giratina Inspired

This set is inspired by the legendary Pokemon Giratina, the palette being the same colors used by this Pockett Monster, as well as using the various design elements found within the design.
